# CV Portfolio Learning Project

This project is a personal CV/portfolio website built as a learning exercise to practice front-end development with HTML, CSS, and JavaScript. The main goal is to improve skills in building modern, responsive, and visually appealing web pages.

## Features

- A clean and stylish personal portfolio layout
- Responsive design for different screen sizes
- Smooth animations and visual effects
- Sections for introduction, contact information, personal details, education, interests, and skills
- Modern icon and font integration
- Reusable styling with CSS variables

## What I Have Done in This Project

- Structured the page using semantic HTML
- Styled the layout with modular CSS files
- Added animation effects to improve user experience
- Integrated external icons, fonts, and libraries
- Built a polished presentation for personal information and skills

## Project Structure

- index.html: main page structure
- css/: stylesheet files
- js/: JavaScript files
- asset/: images and icons

## Technologies Used

- HTML5
